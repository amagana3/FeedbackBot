# FeedbackBot
A discord bot enforcing users to leave feedback before submitting their own tracks for feedback.

## Supported formats:
- Links of the following:
    - Soundcloud
    - Dropbox
- Attachments of the following:
    - .mp3
    - .mp4a
    - .wav
    - .flac
